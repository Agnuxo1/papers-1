## Abstract

This research investigates the computational modeling of semantic interpretation within the Minimalist Program framework, addressing the persistent challenge of mapping syntactic structures to logical forms. We develop a computational implementation of the Transfer Theory, proposing a formal system that parses syntactic derivations and generates corresponding semantic representations using type-theoretic semantics. Our findings suggest that semantic interpretation can be formally characterized as a recursive function operating on syntactic trees, with feature structures serving as the interface between syntax and semantics. The results contribute to the ongoing debate about the nature of the syntax-semantics interface and provide computational tools for testing theoretical predictions about semantic composition.

## Introduction

The relationship between syntax and semantics has long been central to linguistic theory, with generative grammar proposing various interface mechanisms for semantic interpretation. Chomsky's (1995) Minimalist Program, while focused on syntactic economy, leaves the precise nature of the syntax-semantics mapping underspecified. This research addresses this theoretical gap by developing a computational model that operationalizes the Transfer Theory's predictions about semantic composition.

The motivation for this work stems from two persistent challenges in formal semantics: first, the difficulty of modeling semantic composition in a computationally tractable way that preserves linguistic generalizations; second, the need for empirical validation of theoretical claims about semantic interpretation. Recent advances in type-theoretic semantics (e.g., Jacobson 2019; Barker 2021) have provided new formal tools that allow for more precise modeling of semantic phenomena, while advances in computational linguistics have enabled empirical testing of these models.

This research makes three concrete contributions: (1) a formal specification of the syntax-semantics interface in Minimalist terms; (2) a computational implementation that tests the predictions of the Transfer Theory; (3) empirical validation across diverse syntactic constructions. We build on the work of Heim and Kratzer (1998) in type-theoretic semantics, incorporate insights from recent computational approaches (e.g., Kubota 2022), and extend the theoretical framework to handle cross-linguistic variation.

## Limitations and Future Work

This study has several limitations. First, our model assumes a relatively flat semantic representation without explicit treatment of discourse relations. Second, the implementation focuses primarily on English syntax, with limited testing of cross-linguistic phenomena. Third, the computational model does not yet incorporate pragmatic factors that influence semantic interpretation.

Future research should address these limitations by: (1) extending the model to handle discourse-level semantics; (2) testing the framework on typologically diverse languages; (3) incorporating pragmatic inference mechanisms; (4) exploring the relationship between syntactic movement and semantic scope.
